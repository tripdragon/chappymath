θ Θ Theta

_0 _o _O Greek letter θ (theta). Symbol used in sin cos tan etc as a measured angle.

sin(θ) sin(1.2) sin(π)

You can use anything, its just typically used

Sine sin(θ)

_sin Gives an oscillating value between 0 to 1 to 0 to -1 to 0

You tend to feed it and its cousin Cosine _cos an increasing value like Time or _PI π

When the value reaches PI π you have traversed 0 to 1 to 0

One of the most common uses of the value is to assign a 2d/3d object its position y value.

Math.sin(0.1)

        var count = 20;
        var ratio = Math.PI * 2 / count;
        
        for (var i = 0; i < count+1; i++) {
          var theta = ratio * i;
          console.log("theta", theta, " : sin", Math.sin(theta));
        }

Cosine cos(θ)

_cos Gives an oscillating value between 1 to 0 to -1 to 0 to 1

Used the same way as sin(θ)

When the value reaches PI π you have traversed 1 to 0 to -1

Same as sin in 2d/3d object but for position x value.

Math.cos(0.1)

        var count = 20;
        var ratio = Math.PI * 2 / count;
        
        for (var i = 0; i < count+1; i++) {
          var theta = ratio * i;
          console.log("theta", theta, " : sin", Math.cos(theta));
        }
